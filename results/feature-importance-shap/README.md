# Feature Importance & SHAP Analysis

This folder contains the feature importance and SHAP analysis results generated from the selected final XGBoost model.

These outputs support the interpretation of the model by identifying the most influential features contributing to the municipality-level dengue risk classifications.

The analyses are generated within the `Thesis (Final Model with Classification).ipynb` notebook located in the `data-modeling/` folder. This folder stores only the exported result files.
