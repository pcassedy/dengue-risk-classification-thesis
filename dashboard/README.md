# Dashboard

This folder contains resources related to the interactive Looker Studio dashboard developed for the thesis. The dashboard presents the municipality-level dengue risk classification results for CALABARZON and provides interactive visualizations to support data exploration and interpretation.

## Dashboard Features

* Municipality-level dengue risk classification map
* Risk classification summaries and key metrics
* Interactive filters for exploring municipalities and years
* Feature importance and SHAP analysis visualizations
* Supporting charts and tables for model outputs
